LANDASAN TEORI

POINT OF SALE (POS)

Sistem terkomputerisasi yang mengintegrasikan hardware dan software untuk memproses transaksi penjualan, mengelola inventaris, dan menghasilkan laporan keuangan secara real-time.

OOP & MVC

OOP: Paradigma pemrograman berorientasi objek dengan prinsip enkapsulasi, inheritance, polimorfisme, dan abstraksi untuk kode yang modular.

MVC: Pola desain yang memisahkan Model (data), View (tampilan), dan Controller (logika) untuk aplikasi yang terstruktur.

LARAVEL FRAMEWORK

Framework PHP open-source yang mengikuti pola MVC, menyediakan sintaks elegan, fitur lengkap (routing, authentication, caching), serta efisiensi waktu dan kode untuk pengembangan aplikasi web modern.

AUTHENTICATION & AUTHORIZATION

AUTHENTICATION

Menggunakan Laravel Breeze untuk sistem login yang aman:

  • Login system dengan validasi email/password
  • Session management terenkripsi
  • Password hashing dengan bcrypt
  • Remember me functionality

AUTHORIZATION

Role-Based Access Control dengan middleware custom:

// Middleware CheckRole
if (!in_array(auth()->user()->role, $roles)) {
    abort(403, 'Unauthorized access');
}
